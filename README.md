# icarus.finance dApp

## Quick Start

1. Installing dependencies
`yarn install`

2. Development environment
`yarn start`

3. Packing Code
`yarn build`

4. Deployment Code
Use the `build` folder generated in step 4 to place on a server such as nginx, apache, etc.
